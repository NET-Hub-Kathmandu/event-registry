# event-registry

This repository serves as a central archive for all .NET Hub Kathmandu meetups and community events. It contains event assets, presentations, marketing materials, attendee names, photos, recordings, and reports to help preserve the history and impact of our community.

**Purpose**

The goals of this repository are:

Maintain a historical record of all .NET Hub Kathmandu events.
Centralize event assets and documentation.
Enable knowledge sharing among organizers and community members.
Simplify event planning and handover for future organizers.
Preserve presentations and learning materials for attendees.

Organizers/Members are encouraged to upload all event-related assets shortly after each meetup to ensure the repository remains complete and up-to-date.

**About .NET Hub Kathmandu**

.NET Hub Kathmandu is a community-driven platform for .NET developers, software engineers, students, and technology enthusiasts to learn, share knowledge, and connect with fellow professionals.
